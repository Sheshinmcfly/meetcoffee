# Meet & Coffee

Landing page para comunidad de charlas y eventos tecnológicos en Santiago.

## Tecnologías

- Bootstrap 5.3
- Bootstrap Icons 1.11
- HTML5 semántico

## Páginas

- `index.html` — página principal con hero, ubicación, próxima charla, eventos anteriores y contacto
- `evento01.html` — Simposio Vegan DB
- `evento02.html` — Machine Learning 101
- `evento03.html` — Scrum sin Scream
